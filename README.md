# gof
Gang of Four(Java Design Pattern)

- Creational
  - Factory
  - AbstractFactory (with FactoryGenerator, FactoryProducer)
  - Singleton & LazyHolder
  - Builder
  - Prototype (Cloneable)
- Structual
  - Adapter
  - Bridge
  - Decorator (Wrapper)
  - Filter
  - Composite
  - Facade
  - Proxy
  - Flyweight (compared with Prototype)
- Behavioral
- J2EE
